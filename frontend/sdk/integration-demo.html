Basic Drop-in

Add one script tag. HumanGuard automatically scores this session after 30 seconds of behavioral signal collection. Watch the live status below change once scored.

Live Demo — This Page

Collecting signals…
HTML 
<!-- 1. Add this to your <head> or before </body> -->
<script src="https://d1hi33wespusty.cloudfront.net/sdk/humanGuard.min.js"
        data-api-key="hg_live_XXXX"></script>

<!-- That's it. Score is automatically collected after 30 seconds. -->

<!-- Optional: listen for the result -->
<script>
  document.addEventListener('humanGuard:scored', function(e) {
    console.log('Score:', e.detail.prob_bot, e.detail.label);
  });
</script>

Form Protection

Block form submission when prob_bot > 0.8. Try submitting — the SDK scores immediately on submit and blocks bots before the form data is sent.

Contact Form Demo

HTML + JavaScript 
<script src="https://d1hi33wespusty.cloudfront.net/sdk/humanGuard.min.js"
        data-api-key="hg_live_XXXX"
        data-auto-score="false"></script>
<!-- data-auto-score="false" disables the 30s timer; we score on submit -->

<form id="myForm">
  <input type="email" placeholder="Email" required />
  <button type="submit">Submit</button>
</form>

<script>
  const form = document.getElementById('myForm');

  form.addEventListener('submit', async function(e) {
    e.preventDefault();

    // Trigger scoring and wait for result
    window.HumanGuard.score();
    await new Promise(resolve =>
      window.HumanGuard.onScore(resolve)
    );

    const result = window.HumanGuard.getScore();
    if (result.prob_bot > 0.8) {
      document.getElementById('msg').textContent =
        'Submission blocked: suspicious activity detected.';
      return;
    }

    // Safe — proceed with real submission
    form.submit();
  });
</script>

<div id="msg"></div>

Custom Callback & Content Gating

Show premium content only to verified humans. The callback fires the moment scoring completes — use onScore() to update your UI in real time.

Premium Content Gate

🔒
Verifying you're human…
Results in ~30s

✓ Human Verified

Welcome! Your exclusive content is now unlocked.

⚠️
Bot activity detected
Access to premium content restricted.
HTML + JavaScript 
<script src="https://d1hi33wespusty.cloudfront.net/sdk/humanGuard.min.js"
        data-api-key="hg_live_XXXX"></script>

<div id="locked">🔒 Verifying…</div>
<div id="premium" style="display:none">
  Premium content here!
</div>

<script>
  window.HumanGuard.onScore(function(result) {
    const isHuman = result.label === 'human';

    document.getElementById('locked').style.display  = 'none';
    document.getElementById('premium').style.display = isHuman ? 'block' : 'none';

    if (!isHuman) {
      document.body.insertAdjacentHTML('beforeend',
        '<p style="color:red">Bot activity detected.</p>'
      );
    }
  });
</script>

SDK Reference

Attribute / Method Type Description
data-api-keystringRequired. Your hg_live_* API key.
data-thresholdfloatBot probability cutoff (default: 0.5).
data-auto-scorebooleanAuto-score after delay (default: true).
data-delayintegerAuto-score delay in ms (default: 30000).
data-api-urlstringOverride the HumanGuard API base URL.
HumanGuard.getScore()functionReturns current score object or null.
HumanGuard.onScore(cb)functionRegister callback; fires immediately if already scored.
HumanGuard.score()functionTrigger scoring immediately. Idempotent.
HumanGuard.sessionIdstringCurrent session identifier.
humanGuard:scoredeventFired on document with e.detail = {prob_bot, label, threshold, session_id}.
humanGuard:erroreventFired on document with e.detail = {error} on failure.
